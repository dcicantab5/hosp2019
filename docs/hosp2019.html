Specialist Excesses and Shortages:


Explainer:

Based on formula for standardised workload (Std Workload),

\[\begin{equation} {Std\;Workload = {Number\;of\; specialists} \div \left( \displaystyle \frac{Admissions}{1000}\right) } \tag{1} \end{equation}\]



e.g. by 26 May 2022, for Hospital Umum Sarawak which has 51 specialists sharing admissions workload of 12,768 admissions,
\[{{Std\;Workload}_{Current} = {51} \div \left( \displaystyle \frac{12768}{1000}\right) = 4.0\;spec\;per\;1000\;admissions}\]

rearranging (1), we can calculate the number of specialists at median standardised workload for every hospital, given the number of admissions in each hospital

\[\begin{equation} {{Number\;of\; specialists}_{Median} = {Std\;Workload}_{Median} \times \left( \displaystyle \frac{Admissions}{1000}\right) } \tag{2} \end{equation}\]



by 26 May 2022, the median of standardised workload for state hospitals cohort is 2.1 spec per 1000 admissions, hence for Hospital Umum Sarawak,


\[\begin{equation} {{Number\;of\; specialists}_{Median} = 2.1 \times \left( \displaystyle \frac{12768}{1000}\right) = 27\;specialists } \end{equation}\]


hence,

\[\begin{equation} {Adequacy = {Number\;of\;specialists}_{Current} - {Number\;of\;specialists}_{Median}} \tag{3} \end{equation}\]


\[{Adequacy = {51} - {27} = Excess\;24\;specialists}\]

Figure 6: Specialist Excesses and Shortages. Negative values are shortages, and positive values are excesses. The number in shortage/excess is then derived using that hospital’s current workload.
